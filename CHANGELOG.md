# Changelog

## 1.0.0

- First stable release.
- Metrics for lights, switches, media players, and motion binary sensors.
- Measurement sensor statistics: minimum, maximum, average, samples, samples this hour, average samples per hour.
- Runtime tracking by media source.
- Device assignment modes: separate HA-Metric device or source device attachment.
- Display area modes: sensor or diagnostic.
- Update modes: normal, live, custom.
- English and German translations.
